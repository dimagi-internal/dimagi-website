Naatal Mbay: CommCare for Agricultural Support

How the Naatal Mbay program used CommCare to build CommAgri, a digital tool for agricultural extension and stronger value chains for smallholder farmers across Senegal.

D
Dimagi Oct 2019 3 min read Case Study
Share LinkedIn X

Extension agents in Senegal used CommAgri, built on CommCare, to register and support smallholder farmers.

Naatal Mbay, meaning “making agriculture prosperous,” is a Feed the Future project funded by USAID and implemented by RTI International. The program used CommCare to build CommAgri, a digital tool that registered 68,000 farmers and coordinated 500 extension agents across Senegal’s smallholder rice, maize, and millet value chains.

A digital platform for Senegal’s smallholder value chains

Naatal Mbay works to empower farmer-serving organizations active in the rice, maize, and millet value chains across Senegal, giving cooperatives and extension networks the digital tools to operate at scale.

The program built CommAgri, its core digital tool, on CommCare. CommAgri lets extension agents track agricultural inputs and productivity, manage seasonal loans, and collect rainfall data from the field, all offline and all on a single mobile platform that syncs when connectivity is available. By the time of reporting, CommAgri had been adopted by 55 cooperatives and 500 extension agents, who had collectively registered 68,000 farmers across the program’s target value chains.

Three problems that paper couldn’t solve

Before CommAgri, the program faced three connected obstacles that paper-based records could not address:

  • Farmer visibility. Extension agents managed hundreds of farming households across dispersed villages. Without digital records, visit frequency and input adoption were impossible to verify or improve.
  • Cooperative finance. Seasonal loan management, including disbursement, repayment tracking, and advance calculation, required accurate farmer-level records that paper registers could not reliably maintain.
  • Connectivity gaps. Senegal’s agricultural zones have limited or no mobile data coverage, so any digital solution had to work entirely offline and sync data when connectivity returned.

CommAgri: one platform, three programs

The program configured CommCare’s case management, supply tracking, and geographic tools into a unified field platform for extension, finance, and logistics:

  • Farmer registration & visit tracking: agents registered each farming household as a case, capturing household composition, plot location, and crop mix, then logged subsequent visits against the same record to give supervisors a complete coverage history.
  • Input & productivity tracking: agents recorded seed, fertilizer, and input deliveries at the farm level and logged yield estimates at harvest, creating a seasonal productivity record for each household.
  • Loan management: CommCare Supply was configured to manage seasonal loan disbursements and repayments, with automatic calculation logic and SMS confirmations for farmers.
  • Rainfall data collection: a lightweight rainfall module let agents log local precipitation data, helping program managers track water stress patterns across target zones.
  • Area Mapper: CommCare’s geographic tools mapped cooperative zones and plot boundaries, giving program staff spatial oversight of coverage.

68,000 farmers, one digital platform

By bringing extension, finance, and logistics onto one offline-first platform, Naatal Mbay turned scattered paper registers into a connected record of its work in the field. The program reached:

  • 68,000 farmers registered in CommAgri across Senegal’s smallholder value chains.
  • 500 extension agents using CommCare in the field to conduct visits and collect data.
  • 55 cooperatives integrated into the CommAgri digital platform.
Farmers registered

CommAgri, built on CommCare, registered 68,000 farmers across Senegal’s rice, maize, and millet value chains, giving cooperatives and extension agents a single offline-first record of their work in the field.

With farmer-level data flowing from dispersed villages back to program managers, Naatal Mbay could verify extension coverage, manage seasonal finance more accurately, and track productivity across the value chains it set out to strengthen.
